# 📂 Sohee's Portfolio

안녕하세요! 3D 웹 인터랙티브와 사용자 경험을 중요하게 생각하는 **풀스택 개발자 박소희**입니다.  
이곳은 저의 프로젝트와 기술 스택을 정리한 포트폴리오 저장소입니다.

## 🚀 배포 링크 (Live Demo)
👉 [포트폴리오 보러가기](https://sohee9010.github.io) _(배포 예정)_

## 🛠️ 기술 스택 (Tech Stack)

### Frontend & Interactive
- **Core:** React, Next.js, TypeScript
- **Styling:** Tailwind CSS, Vanilla CSS
- **3D Graphics:** Three.js
- **Animation:** CSS Animations, Intersection Observer API

### Mobile Development
- **Android:** Kotlin, Java
- **IDE:** Android Studio

### Backend & Tools
- **Backend:** Node.js, Python, C/C++
- **Database/Cloud:** Firebase
- **Version Control:** Git, GitHub
- **Design:** Figma, Photoshop (GTQ 1급)
- **AI Tools:** AI Utilization (Google AI Essentials)

## ✨ 주요 기능

### 💫 반응형 웹 디자인
- PC, 태블릿, 모바일 어디서든 최적화된 레이아웃
- 유연한 그리드 시스템과 미디어 쿼리 활용

### 🎨 3D 인터랙티브 경험
- Three.js를 활용한 동적 3D 구체 애니메이션
- 마우스 움직임에 반응하는 인터랙티브 요소
- 부드러운 CSS 트랜지션과 애니메이션

### 📱 프로젝트 쇼케이스
- **3D Coverflow 슬라이드:** 프로젝트를 3D 효과와 함께 탐색
- **그리드/슬라이드 뷰 전환:** 사용자 선호에 따른 뷰 모드 제공
- **상세 모달:** 각 프로젝트의 세부 정보, 기술 스택, 성과 확인

### 🎯 세련된 UI/UX
- Glass morphism 디자인의 Contact 섹션
- Aurora 효과를 활용한 배경 그라데이션
- 타입라이터 효과로 동적인 첫인상 연출
- 스크롤 진행률 표시 및 부드러운 섹션 전환

### 📊 타임라인 기반 경력 정리
- 시각적 타임라인으로 교육 및 경력 이력 표시
- 자격증 및 수상 경력 카드 형태로 정리

## 🚀 로컬 실행 방법

```bash
# 저장소 클론
git clone https://github.com/sohee9010/portfolio.git

# 디렉토리 이동
cd portfolio

# 로컬 서버 실행 (Python 사용)
python -m http.server 8000

# 브라우저에서 열기
# http://localhost:8000
```
